# Security Policy

## Supported Versions

We currently support the latest major version of Navigatr. Older versions prior to 5.x do not receive security updates.

| Version | Supported |
|--------|----------|
| 5.x.x | :white_check_mark: |
| 4.x.x | :white_check_mark: |
| < 4.0 | :x: |

## Reporting a Vulnerability

If you discover a security issue in this repository, please report it privately using the "Report a vulnerability" feature on GitHub or by emailing our team at security@example.com. Include a clear description and any steps required to reproduce the issue so we can investigate quickly.

We will acknowledge your report, keep you informed as we work to resolve the issue, and coordinate disclosure. Please give us a reasonable time to address the vulnerability before sharing details publicly.
